# bootstrap-controls-lab

The site presents a **Bootstrap Controls Lab** theme and demonstrates the use of **Bootstrap components including modals, popovers, carousel, tooltips, accordion, and responsive layout classes** to create an interactive single-page application.

---

## Page Overview
- **Bootstrap Controls Page**:  
  Includes a responsive single-page Bootstrap layout featuring:
  - A hero-style header introducing Bootstrap
  - A left column with interactive buttons using popovers and a modal
  - A middle column with a responsive carousel and tooltip-enabled elements
  - An email input form with icons and tooltips
  - A right column with an accordion displaying recipe content

Each section demonstrates a different Bootstrap control and how it can be used to enhance user interaction and layout responsiveness.

## Assignment Objectives
This project demonstrates:
- Creating a responsive HTML page using Bootstrap
- Implementing a Bootstrap modal triggered by a button
- Using Bootstrap popovers on interactive buttons
- Building a responsive carousel with indicators and controls
- Adding Bootstrap tooltips to images and form elements
- Creating a responsive accordion with collapsible sections
- Structuring a multi-column responsive layout using Bootstrap grid classes
- Applying custom CSS for layout refinement and visual consistency
